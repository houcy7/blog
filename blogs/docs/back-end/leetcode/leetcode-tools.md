## 目的
为了提高刷题的效率，并对刷题过程进行一个记录，开发了此工具（可能只适用于我，大部分路径都是写死的QAQ）

## 功能
通过一个命令对需要编写的题目进行下载并生成描述md和解题代码，解题完成后补全解题过程并通过脚本生成解题笔记，
然后将解题笔记推送到自己的博客上，完成"闭环"。
